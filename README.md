# Personal site — Ling Cheng

Minimal static site. Replace the headshot at `assets/headshot.jpg` with your photo.

Deploy (GitHub Pages):

1. Create a repo and push this folder to `main` (or use `username.github.io`).
2. In GitHub repo Settings → Pages, select the `main` branch as the source.

Or serve locally:

```bash
python3 -m http.server 8000
# then open http://localhost:8000
```
